###游戏说明：
纸牌匹配游戏，有2张牌和3张牌两种模式，玩家翻开2张或3张牌后，如果期中有两张红色或者点数相同则认为该组纸牌匹配成功，并获得相应的分数，如果没有匹配成功则扣除相应的分数。
